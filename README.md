# ▶️ How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Import the MySQL database into phpMyAdmin.
4. Copy the project folder to:

```
C:\xampp\htdocs\
```

5. Open your browser and visit:

```
http://localhost/apex_planet_intern/
```
