# Task 2 - User Authentication System

## 📌 Project Overview
This project is developed as part of the ApexPlanet Full Stack Development Internship (Task 2).

The project demonstrates a complete User Authentication System using PHP, MySQL, Bootstrap 5, JavaScript, and AJAX.

---

## 🚀 Features

- User Registration
- User Login
- User Logout
- Session Management
- Password Hashing
- Duplicate Email Validation
- AJAX Email Availability Check
- Show/Hide Password
- Responsive Login Page
- Responsive Registration Page
- Bootstrap 5 UI
- Custom CSS Styling

---

## 🛠 Technologies Used

- HTML5

- AJAX

## ⚙️ How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Import the database into phpMyAdmin.
4. Copy the project into the htdocs folder.
5. Open the browser and run:

```
http://localhost/apex_planet_intern/task2/login.php
```
