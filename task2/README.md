# Task 2 - User Authentication System

## Overview

This project is a responsive User Authentication System developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript as part of my Full Stack Development Internship.

## Features

- User Registration
- User Login
- Password Hashing
- Password Verification
- Duplicate Email Validation
- Session Management
- Dashboard
- Logout
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- MySQL

## Database

Database Name

```
login_system
```

Table Name

```
users
```

Columns

- id
- name
- email
- phone
- password

## How to Run

1. Start Apache and MySQL using XAMPP.
2. Import or create the `login_system` database.
3. Create the `users` table.
4. Copy the project into the `htdocs` folder.
5. Open:

```
http://localhost/apex_planet_intern/task2/login.php
```
