# 🎓 CourseHub

A Full Stack Web Application developed using PHP, MySQL, Bootstrap, and PHPMailer.

This project was developed as part of a **Full Stack Development Internship** to demonstrate authentication, role-based access control, database management, and CRUD operations.

---

# 📌 Features

## Authentication

- User Registration
- Email OTP Verification
- Secure Login
- Logout
- Password Hashing
- Role-Based Authentication

---

## Student Module

- Student Dashboard
- View Available Courses
- Enroll in Courses
- My Courses
- Student Profile

---

## Admin Module

- Admin Dashboard
- Dashboard Statistics
- View Registered Students
- View Student Enrollments

# 🚀 Installation

1. Install XAMPP

2. Copy project into

```
htdocs/
```

3. Import

```
coursehub_db.sql
```

using phpMyAdmin.

4. Start

- Apache
- MySQL

5. Open

```
http://localhost/CourseHub/
```
