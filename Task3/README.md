# Task 3 - User Management System

## Overview
This project is a User Management System developed using PHP, MySQL, Bootstrap 5, HTML, CSS, and JavaScript. It allows authenticated users to manage user records through a responsive dashboard.

## Features
- User Login Authentication
- Session Management
- Dashboard
- Add New User
- Edit User Details
- Delete User
- Search Users
- Pagination
- Total Users Card
- Responsive Bootstrap UI

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## How to Run
1. Start Apache and MySQL in XAMPP.
2. Import the database into phpMyAdmin.
3. Place the project inside `htdocs`.
4. Open:
```
http://localhost/apex_planet_intern/Task3/login.php
```
